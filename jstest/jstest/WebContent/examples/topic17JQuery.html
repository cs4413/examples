JQuery fundamentals

JQuery is a widely used library to make AJAX easier. JQuery has:

  • HTML/DOM manipulation
  • CSS manipulation
  • HTML event methods
  • Effects and animations
  • AJAX
  • Utilities

Installation

To install, download JQuery from http://jquery.com/download/ and put it in your project. (Make a js directory under WebContent.) Also make a tests subdirectory and download qunit-1.15.0js and qunit-1.15.0.css from http://qunitjs.com/.

JQuery syntax

An excellent reference is the jQuery API: http://api.jquery.com/. jQuery is based on performing actions on selected HTML elements.

  • $ designates jQuery
  • Syntax is: $(selector).action()
  • this refers to the current selected element.
  • $(document).ready(fn(){ }); installs a jQuery function fn after document has finished loading
  • The jQuery box model is a little different than CSS: http://www.w3schools.com/jquery/jquery_dimensions.asp

JQuery selectors

  • By element: $("p")
  • By id: $("#test")
  • By class: $(".test")
  • Descendents: $("#test p") refers to paragraphs within the element identified by id test.
  • Adjacent siblings: $("h2 + div) refers to div elements that are adjacent siblings of h2.
  • Siblings: $("ul ~ p") refers to all p elements that are siblings of a ul element.
  • Children: $("div > p") refers to all p elements that are children of div elements.

JQuery methods ($selector.method)

  • val gets the value of the selected element.
  • val(mvalue) sets the value to mvalue.
  • text gets the text of the selected element.
  • test(mvalue) sets the text of the selected element to mvalue.
  • next(tvalue) gets the next sibling of selected element of specified type.
  • submit submits the selected form.
  • focus moves focus to selected control or link.
