zRSSFeed - Multiple RSS feeds with ticker animation

This example shows multiple feeds on a single page animated by the vTicker plugin.

Script

$(document).ready(function () {
	$('#ticker1').rssfeed('http://feeds.reuters.com/reuters/peopleNews',{}, function(e) {
		$(e).find('div.rssBody').vTicker();
	});

	$('#ticker2').rssfeed('http://feeds.bbc.co.uk/iplayer/highlights/tv',{}, function(e) {
		$(e).find('div.rssBody').vTicker({ showItems: 1});
	});
});
